### Environment Variable

Create a .env file with values:
```

MONGO_URI=mongodb+srv://dev064:Adityajha111@cluster1.tizixwp.mongodb.net/Kata
```

### Installing Dependencies
````
npm install
````
### Run the Project
````
npm run dev
````

### Run the Test Cases
````
npm test / npm t
````
